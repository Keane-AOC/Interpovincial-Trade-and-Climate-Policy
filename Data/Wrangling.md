# Wrangling Issues
Most of the data I have downloaded appeared to be clean; there were very few null values. The only data set that had null values was the carbon emissions data set, but this was easy to deal with as it was one data point.

Where I ran into issues was connecting and uniting the data. This was particularly difficult when it came to merging all the provincial energy profiles into one folder. I ran into multiple issues in Tableau as the program was not allowing me to make a union of the different data sets. As a result, I ended up putting all the data into one Excel file as it was easier for me to manage. There were additional issues, such as adding values to data, but I again dealt with this issue in Excel, as it was easier.

There is some more data that I need to wrangle together, but I do not think I have enough expertise to do this on my own. My focus for the next couple of days is making the data usable for visualizations in Tableau (e.g. adding years).

My main assumption going into this was that the data would connect/unite better than they did. I believe that the data struggled to unite due to mismatched category titles or other minor issues that are making Tableau struggle.
